📍 Sender / Kitchen Location

Used for: delivery distance, ETA calculation, and the live tracking map kitchen pin.
Format: lat, lng — e.g. 9.91918526788082, 78.09422025024031
To get: Open Google Maps → right-click your location → copy the two numbers shown at top.

🛵 Delivery ETA Configuration

ETA Formula: Total = Prep Time + (Distance ÷ Speed) + Buffer
Example: 25 min + (3 km ÷ 0.3) + 5 = 40 min total
Overridable per item below

📋 UPI Payment Flow

Customer steps:
1. See QR code + UPI ID at checkout
2. Scan using GPay / PhonePe / Paytm
3. Pay exact amount
4. Enter Transaction/UTR ID
5. Order placed — you verify in your UPI app
Refund: Order can't be processed → Admin clicks Refund → Customer notified via SMS.

⚠️ WhatsApp notifications go to the admin number only — never to customer WhatsApp.

📊 Google Sheets Auto-Logging

One-time setup:
1. Create a Google Sheet
2. Extensions → Apps Script → paste webhook handler script
3. Deploy as Web App (Execute as: Me, Access: Anyone)
4. Paste the Web App URL below
